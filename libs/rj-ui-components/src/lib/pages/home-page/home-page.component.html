This is a library/toolkit for complete standalone UI Components written in Angular.

You will also find styling examples for proper implementation and scoping regarding css, scss and less.

How it works

Under src/app there is an angular module called ui-component.module.ts. This module is the exported module which gets packaged and shipped. The app.module.ts is for running the development site(what you are currently looking at).

Add re-usable components to this project and be sure to document them and make them as agnostic as possible. Provide lots of configuration options.

Releasing

When you go to package and ship to local npm, you should version the package.json with semver accurately to represent the depth of changes and if it is breaking to previous versions.

To generate a distributable .targz, run 'npm run release'. Then if your npm registry is set correctly(make sure its not going to the public npm) then run npm publish.( publish can take a registry as an argument).
